Imprimir cada carácter de una cadena

Explicación del Código para Imprimir cada Carácter de una Cadena

Ejercicio#19, muestra cómo recorrer una cadena y acceder a cada uno de sus caracteres:

  1. Qué es una cadena: Una cadena es una secuencia de caracteres, como letras, números o símbolos.
  2. Índices en cadenas: Cada carácter en una cadena tiene una posición específica, empezando desde 0.
  3. Bucle for: Un bucle for nos permite repetir una acción varias veces. Aquí lo usaremos para recorrer cada carácter de la cadena.
Miembros: Wimper Zamora, Arianna Mosquera, Andres Toscano.
