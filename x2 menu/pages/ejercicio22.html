Contar palabras en una cadena

Explicación del Código para Contar Palabras en una Cadena

Ejercicio#22, cuenta las palabras en una cadena considerando que las palabras están separadas por espacios:

  1. Separación por Espacios: Explicamos que en una cadena, las palabras están separadas por espacios en blanco (" ").
  2. Contador de Palabras: Se utiliza la variable contadorPalabras para contar las palabras encontradas. Inicialmente se asume que hay al menos una palabra (contadorPalabras = 1).
  3. Bucle for para Recorrer la Cadena: Se utiliza un bucle for para recorrer cada carácter de la cadena y verificar si es un espacio.
  4. Incrementar el Contador: Si el carácter actual es un espacio, se incrementa el contador de palabras (contadorPalabras++).
