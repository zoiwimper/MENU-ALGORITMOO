Presentar los nombres que no sean "Ana" de un arreglo

Explicación del Código para Mostrar Nombres que no sean "Ana"

Ejercicio#17, muestra los nombres que no sean "Ana" de un arreglo siguiendo estos pasos:

  1. for (let i = 0; i < nombres.length; i++): Itera a través del arreglo nombres.
  2. if (nombres[i] !== "Ana"): Verifica si el nombre en la posición actual del arreglo no es "Ana".
Miembros: Wimper Zamora, Arianna Mosquera, Andres Toscano.
