Invertir un número dividiendo para diez

Explicación del Código para Invertir Números

Esta función ejercicio #10, realiza lo siguiente:

  1. while (numero !== 0): Itera mientras numero no sea 0.
  2. let digito = numero % 10: Obtiene el último dígito del número.
  3. invertido = invertido * 10 + digito: Agrega el dígito al número invertido.
  4. numero = Math.floor(numero / 10): Elimina el último dígito del número original.
Miembros: Wimper Zamora, Arianna Mosquera, Andres Toscano.
