El promedio de los números de un arreglo

Explicación del Código para Calcular el Promedio

ejercicio #15,calcula el promedio de los números de un arreglo siguiendo estos pasos:

  1. for (let i = 0; i < numeros.length; i++): Itera a través del arreglo numeros.
  2. suma += numeros[i]: Suma cada número al total.
  3. let promedio = suma / numeros.length: Calcula el promedio.
Miembros: Wimper Zamora, Arianna Mosquera, Andres Toscano.
