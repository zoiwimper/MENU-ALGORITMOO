Verificar si un número es primo

Explicación del Código

Esta función ejercicio #9 ealiza lo siguiente:

  1. Obtiene el valor ingresado por el usuario en el campo de texto con el id "num".
  2. Convierte el valor ingresado a un entero usando parseInt().
  3. Verifica si el número ingresado es válido y mayor que 1.
  4. Verifica si el número es primo comprobando si es divisible por cualquier número menor que él mismo (excluyendo 1).
  5. Muestra el resultado correspondiente.
