Los pares de un arreglo recorridos desde el último elemento al primero

Explicación del Código para Mostrar Números Pares

Ejericio#18, muestra los números pares de un arreglo recorriéndolo desde el último elemento al primero siguiendo estos pasos:

  1. for (let i = numeros.length - 1; i >= 0; i--): Itera desde el último elemento al primero del arreglo numeros.
  2. if (numeros[i] % 2 === 0): Verifica si el número en la posición actual del arreglo es par.
Miembros: Wimper Zamora, Arianna Mosquera, Andres Toscano.
