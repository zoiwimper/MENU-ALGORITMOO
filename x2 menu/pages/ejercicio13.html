Presentar la serie de Fibonacci de n números

Explicación del Código para la Serie de Fibonacci

Esta función ejercicio #13, realiza lo siguiente:

  1. console.log(a) y console.log(b): Imprimen los dos primeros números de la serie.
  2. for (let i = 2; i < n; i++): Itera desde 2 hasta n - 1.
  3. let c = a + b: Calcula el siguiente número de la serie.
  4. a = b y b = c: Actualizan los valores para la siguiente iteración.
Miembros: Wimper Zamora, Arianna Mosquera, Andres Toscano.
