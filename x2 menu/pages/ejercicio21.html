Invertir una cadena

Explicación del Código para Invertir una Cadena

Ejercicio#21,invierte una cadena utilizando un bucle for para recorrer la cadena desde el último carácter hasta el primero:

  1. Acumulación de Resultados: Se utiliza la variable cadenaInvertida para acumular los caracteres de la cadena en orden inverso.
  2. Recorrido Inverso: El bucle for for (let i = cadena.length - 1; i >= 0; i--) inicia desde el último carácter de la cadena (cadena.length - 1) y decrementa i hasta llegar a 0. En cada iteración del bucle, se concatena (+=) el carácter actual de cadena a cadenaInvertida, logrando así construir la cadena invertida.
