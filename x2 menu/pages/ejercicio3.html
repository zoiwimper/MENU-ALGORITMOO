Determinar si un número es positivo, negativo o cero

Explicación del Código

Este ejercicio 3 realiza lo siguiente:

  1. Obtiene el valor ingresado por el usuario en el campo de texto
  2. Convierte el valor ingresado a un entero
  3. Obtiene la referencia al elemento, donde se mostrará el resultado.
  4. Verifica si el número es positivo (> 0), negativo (< 0) o cero (== 0).
  5. Muestra el resultado correspondiente en el elemento
