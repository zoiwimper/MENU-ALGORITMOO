El mayor de los elementos de un arreglo

Explicación del Código para Encontrar el Mayor

Ejercicio#16,encuentra el mayor de los elementos de un arreglo siguiendo estos pasos:

  1. let mayor = numeros[0]: Supone que el primer elemento es el mayor.
  2. for (let i = 1; i < numeros.length; i++): Itera desde el segundo elemento.
  3. if (numeros[i] > mayor): Actualiza mayor si se encuentra un número más grande.
Miembros: Wimper Zamora, Arianna Mosquera, Andres Toscano.
